About Our Game

GenericRPG is a text-based fantasy role-playing game.
It can be played using any modern web browser, such as Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge.
Simply visit the the login page and create an account.

Character creation is quick and easy!

Our goal was to create an online gaming communications framework to allow multiple users to play at the same time!

Want to build your own game? We can help! Just check out our GitHub!
